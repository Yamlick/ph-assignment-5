Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface by home java script can easily manipulates Html elements .

How do you select an element from the DOM?

There are 4 different ways

1.getElementById('id')
2.getElementByClassName("class") 3.getElementByTagName("tag") 4.querySelector('selector') and querySelectorAll('selector')

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a powerful technique in java script parent node handles event so it is not necessary to attach event in every root level.so this helps to acheive efficiency

How do you manipulate an element's attributes and styles using the DOM?

manipulate attribute: setAttribute(name, value), getAttribute(name), removeAttribute(name)
manipulate styles : element.style.backgroundColor = "tomato";
